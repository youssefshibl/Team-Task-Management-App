# Quick Start Guide

Get up and running in 3 easy steps!

## Step 1: Install Dependencies

```bash
cd frontend
npm install
```

## Step 2: Start Backend

Make sure the backend is running on port 3000:

```bash
cd ../backend
npm install
npm run start:dev
```

## Step 3: Start Frontend

```bash
cd ../frontend
npm run dev
```

Open your browser to `http://localhost:5173`
